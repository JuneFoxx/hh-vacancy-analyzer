# Vacancy Analyzer HH.ru

**Инструмент для анализа рынка труда с HH.ru с использованием ИИ**

Telegram-бот, который помогает быстро понять, какие навыки сейчас востребованы, показывает статистику и среднюю зарплату по запросам.

---

## ✨ Основные возможности

- **Поиск вакансий** по ключевому слову и региону
- **Поиск по навыкам** — находит вакансии, где присутствуют **все** указанные навыки
- **Статистика востребованности** навыков
- **Средняя зарплата** по запросу и региону
- **Автоматический сбор** вакансий (можно запускать по расписанию)
- Анализ навыков с помощью локального ИИ (Ollama)

---

## Как пользоваться

1. Напишите боту `/start`
2. Выберите:
   - «🔍 Новый поиск» — поиск по ключевому слову
   - «🔎 Поиск по навыкам» — поиск по списку навыков
3. Укажите регион и запрос/навыки

---

## Технологии

- Django + Django REST Framework
- aiogram 3 (Telegram Bot)
- Ollama (локальный ИИ)
- HH.ru API
- SQLite
